# clj-varint

Simple wrapper around Bazel VarInt code.
https://github.com/bazelbuild/bazel/blob/master/src/main/java/com/google/devtools/build/lib/util/VarInt.java

## Usage

To install locally:
* Have Boot (boot-clj.com)
* Run `boot build`
